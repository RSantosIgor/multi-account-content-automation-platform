- Regras de prompt é um campo de texto que o usuário pode escrever um prompt para que a AI possa usar como base para gerar o resumo da matéria. O usuário pode escrever várias regras de prompt. Então uma conta pode conter uma lista de regras de prompt. As regras são de dois tipos que devem ser informadas no momento do registro da regra. Os dois tipos são: regras de prompt de para analise e regras de prompt para publicação. As regras se prompt de análise serão usadas para gerar o resumo da matéria e se a matéria deve ser sugerida ou não, pois alguns artigos serão descartados. As regras de prompt de publicação serão usadas para gerar o post que será publicado.
